# DevTools Hub

A collection of free developer tools. Fast, secure, and privacy-focused. Everything runs in your browser.

## Features

- 43+ developer tools
- Bilingual support (EN/RU)
- Dark theme with modern UI
- Client-side processing (no data leaves your browser)
- Mobile-friendly responsive design

## Tool Categories

### Encoding/Decoding
- Base64, Base32, Hex converters
- URL, HTML Entities encoding
- Unicode Escape, ASCII Binary
- Punycode, Quoted-Printable
- ROT13/ROT47, Morse Code, Braille
- NATO Phonetic Alphabet

### Number Systems
- Binary/Decimal/Hex/Octal converter
- Roman Numerals
- Scientific Notation
- IEEE 754 Float Visualizer
- Big Number Calculator (BigInt)
- Bitwise Operations

### Data Formats
- JSON ↔ CSV/TSV
- JSON ↔ Query String, Form Data
- XML ↔ JSON
- Markdown ↔ HTML
- YAML/JSON Schema Generator
- MessagePack ↔ JSON
- BSON ↔ JSON (MongoDB)

### Date & Time
- Unix Timestamp converter
- Date Difference Calculator
- Cron Expression Parser
- Date Format Converter (ISO/RFC)
- Timezone Converter
- Age Calculator
- Working Days Calculator
- Calendar Week Calculator
- Leap Year Checker

### Other Tools
- DNS Health Check
- PostgreSQL Index Advisor
- Image Optimizer (WebP/AVIF)
- Format Converter (JSON/YAML/XML/CSV/TOML)
- CSS Gradient Generator
- UUID Generator
- Hash Generator (MD5, SHA)

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Zustand
